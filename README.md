# Chess Game

## Project Overview
This Chess Game project is a digital implementation of the classic board game Chess, designed to provide an interactive experience for users.

## Features
- Multiplayer support
- AI opponent
- Move validation
- Undo and redo moves
- Save and load game states

## Technologies Used
- Programming Language: Python
- GUI Framework: Tkinter
- Version Control: Git

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/settybalijasiakumar076/chess-game.git
   ```
2. Navigate to the project directory:
   ```bash
   cd chess-game
   ```
3. Run the application:
   ```bash
   python chess_game.py
   ```

## Usage Instructions
- Start a new game by selecting 'New Game'.
- Choose your side: White or Black.
- Make your move by clicking on the pieces and the target squares.
